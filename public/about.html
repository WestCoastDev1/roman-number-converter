About

This Roman number converter has been written for a Coding Kata.

The constraints on time have meant that the result is mainly in vanila JavaScript.

Development is by TDD using mocha / chai.

The application (while relatively trivial) has been treated like production code and

as such validation and securitization (such as try catches where there is liitle real

chance of malfunction) have been added to give it some authenticity.
